Module: backend/src/authentication.js

This file handles user authentication middleware and routes. It exports functions for registering a new user and logging in an existing user. The register function creates a new user account in the PostgreSQL database, hashes the user's password using bcrypt, and sets a JSON Web Token (JWT) as an HttpOnly cookie.
