Module: backend/src/mainAPIRoutes.js

This file exports several route handlers for the Emergent Playground project. These include handlers for creating a new post, downloading post state, downloading post properties, downloading a list of posts, and getting a username. Each handler interacts with the PostgreSQL database to perform its function.
